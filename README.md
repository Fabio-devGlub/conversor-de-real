# 💱 Conversor-de-Real

Projeto de **conversor de moedas** desenvolvido com **HTML, CSS e JavaScript**, que permite converter valores em **Real (BRL)** para **Dólar (USD)**, **Euro (EUR)** e **Libra (GBP)**.

A aplicação busca as cotações atualizadas através de uma API externa e realiza a conversão em tempo real.

---

## 🚀 Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript (ES6+)
* API de cotação de moedas

---

## 🌐 API utilizada

As cotações das moedas são obtidas através da API:

https://docs.awesomeapi.com.br/api-de-moedas

Essa API fornece valores atualizados das moedas em relação ao **Real Brasileiro (BRL)**.

---

## ⚙️ Funcionalidades

* Converter **Real (BRL)** para:

  * 💵 Dólar Americano (USD)
  * 💶 Euro (EUR)
  * 💷 Libra Esterlina (GBP)
* Buscar **cotações atualizadas automaticamente**
* Alterar **bandeira e nome da moeda selecionada**
* Exibir valores formatados corretamente para cada moeda
